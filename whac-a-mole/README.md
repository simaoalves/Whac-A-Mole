# Whac-A-Mole Game

## Running Locally

1. Clone the repository
2. Run `npm install` to install dependencies
3. Run `ng serve` to start the application
4. Open [http://localhost:4200](http://localhost:4200) in your browser to play

## Time Spent

- Design: 1 hour
- Implementation: 2 hours
- Testing: 1 hour
